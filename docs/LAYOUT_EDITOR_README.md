# 保管库3D布局编辑器使用指南

## 概述

保管库3D布局编辑器用于在网页端搭建保管库柜组的三维布局，编辑完成后可在"实时监视"页面加载该布局并实时显示重量与状态。

## 使用流程

### 第一步：创建布局

1. 在左侧导航栏点击"布局编辑器"进入编辑器页面
2. 在左上角"布局管理"区域输入布局名称
3. 点击"新建"按钮创建一个空白布局

### 第二步：添加柜组到场景

1. 在左侧"组件库"区域选择柜组模板（单列/双列/三列/四列/高柜/矮柜）
2. 点击模板右侧的"+"按钮，柜组将被添加到3D场景中
3. 可以重复添加多个柜组

### 第三步：调整柜组位置和参数

1. 在3D画布中点击柜组进行选中（选中后显示蓝色高亮）
2. 使用顶部工具栏切换变换模式：
   - **平移 (G)**：拖拽箭头移动柜组位置
   - **旋转 (R)**：拖拽圆环旋转柜组方向
   - **缩放 (S)**：拖拽手柄调整柜组大小
3. 在右侧"变换属性"面板中精确输入位置坐标和旋转角度
4. 在右侧"模型参数"面板中调整列数、层数、柜宽、柜高、柜深

### 第四步：绑定柜组资产

1. 选中场景中的柜组实例
2. 在右侧"资产绑定"区域点击"绑定柜组资产"按钮
3. 从下拉列表中选择已在"保险柜组"模块中创建的柜组资产
4. 绑定后，实时监视页面将显示该柜组的实时重量数据

注意：同一布局中不允许重复绑定同一个柜组资产。

### 第五步：保存和激活布局

1. 点击"保存"按钮保存当前布局配置
2. 点击"激活"按钮将当前布局设为活跃布局
3. 激活后，"实时监视"页面将自动加载该布局

### 第六步：查看实时监视

1. 点击左侧导航"实时监视"进入监视页面
2. 页面将自动加载已激活的布局，以3D方式展示所有柜组
3. 鼠标悬停在柜组上方可查看详细信息（当前重量、初始重量、变化量、报警阈值）
4. 柜组颜色按状态自动变化：青色=正常、琥珀色=警告、红色=报警
5. 底部统计卡片显示监控总数、正常、警告、报警数量

## 画布交互

| 操作 | 说明 |
|------|------|
| 鼠标左键拖拽 | 旋转视角 |
| 鼠标右键拖拽 | 平移视角 |
| 鼠标滚轮 | 缩放视角 |
| 点击柜组 | 选中柜组 |
| 快捷键 G | 切换到平移模式 |
| 快捷键 R | 切换到旋转模式 |
| 快捷键 S | 切换到缩放模式 |
| Delete 键 | 删除选中柜组 |

## 权限说明

布局编辑器仅对管理员（admin角色）开放。普通用户只能查看实时监视页面。
