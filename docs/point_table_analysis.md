# 称重系统点位表分析

## 设备型号统计
| 型号 | 数量 | 说明 |
|------|------|------|
| DYX-301 | 646 | 称重传感器（惠斯通桥） |
| DYJXH-S4 | 2 | 4路接线盒 |
| DYJXH-S6 | 35 | 6路接线盒 |
| DYJXH-S8 | 56 | 8路接线盒 |
| DY7001 | 11 | 单通道称重仪表（1端子+1个RS485） |
| DY7004 | 22 | 四通道称重仪表（4端子+1个RS485） |

## 硬件链路关系
传感器(A***) → 接线盒(B***) → 称重仪表端子(C***) → 485网关COM端口

## 三个区域
1. **内圈柜组**：9个柜组，使用16口485交换机
   - 内圈1-4柜组 → C001(DY7004) 端子1-4 → COM1
   - 内圈5-8柜组 → C002(DY7004) 端子1-4 → COM2
   - 内圈9柜组 → C003(DY7001) 端子1 → COM3

2. **过道柜组**：多个柜组，使用32口485交换机
   - 过道1柜组 → C004(DY7004) 端子1-4 → COM1（多端子组合）
   - 过道2柜组 → C005(DY7004) 端子1-4 → COM2
   - 过道3柜组 → C007(DY7004) 端子1-4 + C006(DY7001) 端子1 → COM3/COM4

3. **外圈柜组**：多个柜组，使用16口485交换机
   - 外圈1柜组 → C026(DY7004) 端子1-4 → COM4（无接线盒，传感器直连仪表）
   - 外圈2-3柜组 → C027(DY7004) 端子1-4 → COM5
   - 外圈4-5柜组 → C028(DY7004) 端子1-4 → COM6

## 关键发现
1. **一个柜组可能绑定多个通道**：如过道1柜组绑定了C004的端子1+端子2+端子3（3个接线盒的信号）
2. **一个DY7004的4个端子可能分属不同柜组**：如C001的端子1-4分别对应内圈1-4柜组
3. **DY7001只有1个端子**，DY7004有4个端子
4. **每个柜组的重量 = 绑定的所有通道值之和（可带系数）**
5. **接线盒型号决定传感器数量**：S4=4路, S6=6路, S8=8路
6. **同一COM端口下可能有多台仪表**（不同slaveId）
7. **外圈1柜组特殊**：传感器直连仪表端子（无接线盒）
