# ZLAN6808 RS485网关技术规格

## 基本信息
- **产品名称**：ZLAN6808 8路DI/DO/AI远程IO控制器
- **版本**：Rev.2 (ZL DUI 20211008.2.0)
- **支持的通信协议**：RS485、NB、以太网、4G、LoRa、CAT1、Zigbee

## 主要功能特点
1. **8路DI/DO/AI输入输出**
   - DI（数字输入）：支持干节点和湿节点，带光隔离
   - DO（数字输出）：继电器输出，5A 250V AC或5A 30V DC
   - AI（模拟输入）：前4路支持0~5V电压输入，后4路支持4~20mA电流输入
   - ADC精度：12位

2. **RS485接口**
   - 支持Modbus协议
   - 可连接第三方RS485采集器、控制器
   - 用于远程控制

3. **通信功能**
   - 支持Vircom工具连接设备配置
   - Modbus寄存器表用于数据交互
   - 支持DI主动上报、AI主动上报
   - DI和AI同时上报功能

## 关键应用场景
- 称重传感器数据采集（通过AI模拟输入）
- 远程IO控制和监测
- 多种网络接入方式（以太网、4G、LoRa等）

## 集成建议
- 将ZLAN6808作为称重系统的数据网关
- 通过RS485连接多个称重仪表
- 利用网络功能实现远程数据推送
- 支持告警输出和状态监测
